* Zsh with Ohmyzsh (https://ohmyz.sh/#install)
* Hyperland
* Waybar
* Rofi

# Get started
 ```bash
  	sh -c "$(curl -fsSL https://raw.githubusercontent.com/rubiin/dotfiles/master/dot_bin/executable_install-all.sh)"
```
To use chezmoi:
chezmoi -- init --apply rubiin
